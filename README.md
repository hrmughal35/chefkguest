# Chef K Guest — Ultra-Premium Luxury Restaurant Website

**Live at:** `http://localhost:3000`

## About
A complete, production-ready luxury restaurant website for **Chef K Guest**, Gujranwala's premier dining destination.

---

## Tech Stack
- **Next.js 15** — App Router, React Server Components
- **TypeScript** — Fully typed
- **Tailwind CSS** — Custom luxury theme
- **Framer Motion** — Premium animations & transitions
- **Lucide React** — Icon system
- **Google Fonts** — Playfair Display + Inter

---

## Features
- Cinematic hero section with parallax scroll
- Luxury page loader animation
- Sticky transparent navbar with active states
- Brand story section with floating badges
- Experience cards with hover effects
- Animated statistics counters
- 6 signature cuisine cards with hover reveals
- Chef's recommendations with ratings
- Hi-Tea, Platters & Desserts showcase
- Masonry gallery with fullscreen lightbox
- Private events & celebrations section
- Animated testimonial slider
- Luxury reservation form with success state
- Interactive Google Maps embed
- Floating action buttons (WhatsApp, Call, Reserve)
- Scroll-to-top button
- SEO optimized metadata
- Mobile-first fully responsive design

---

## Getting Started

```bash
npm install
npm run dev      # Development server at http://localhost:3000
npm run build    # Production build
npm start        # Start production server
```
